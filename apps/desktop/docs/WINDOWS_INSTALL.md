# GenLoop Desktop Windows 安装指南

## 系统要求

- Windows 10 (64位) 或更高版本
- 4GB RAM (推荐 8GB)
- 500MB 可用磁盘空间

## 安装方式

### 方式一：安装程序 (推荐)

1. 下载 `GenLoop-Desktop-Setup-3.0.0-x64.exe`
2. 双击运行安装程序
3. 选择安装路径 (默认: `C:\Program Files\GenLoop Desktop`)
4. 完成安装

安装程序会自动：
- 创建桌面快捷方式
- 创建开始菜单项
- 注册 `genloop://` 协议

### 方式二：便携版

1. 下载 `GenLoop-Desktop-Portable-3.0.0-x64.exe`
2. 放到任意目录
3. 双击运行，无需安装

## 卸载

### 安装程序版

1. 打开 设置 → 应用 → 应用和功能
2. 找到 "GenLoop Desktop"
3. 点击卸载

或使用开始菜单中的卸载快捷方式。

### 便携版

直接删除程序文件即可。

## 故障排除

### 无法启动

1. 确保已安装 Visual C++ Redistributable
2. 以管理员身份运行
3. 检查杀毒软件是否拦截

### 需要管理员权限

某些功能（如系统托盘、全局快捷键）需要管理员权限。

## 更新

应用会自动检查更新，或从 GitHub 下载最新版本。
